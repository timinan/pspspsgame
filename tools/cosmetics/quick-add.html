Cosmetic Quick Add

Drop one PNG → fully integrated cosmetic in ~30s. Bobs with the cat automatically.
Drop a PNG here, or click to choose a file
91 × 64 pixel-art recommended — alpha-transparent background, item only
What happens on submit:
  1. PNG saved as assets-raw/cosmetic/c<N>/c<N>_idle.png
  2. npm run extract:assets regenerates the atlas + frame-offset data
  3. Catalog entry appended to tools/cosmetics/cosmetics.json
  4. npm run sync:catalog regenerates the typed catalog
  5. Devvit playtest auto-rebuilds — refresh the playtest tab to see it in DressingRoom

Bobs / jumps with the cat via the cat's per-frame translation offsets (cat-frame-offsets.json) — no per-frame art needed.
