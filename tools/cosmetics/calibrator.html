Offset is in cat-source pixels from the cat's vertical midline (X=0) and from the top of the cat sprite (Y=0). Y < 0 sits above the top, Y > 0 sits inside the cat's body.
Check the tints you want to spin off as new cosmetics from c1. Click anywhere on a row to preview that tint on the canvas. Click Generate to add each checked variant as a real catalog entry (c18, c19, …) — they inherit slot, offset, scale, and rarity from this base, and you can keep editing them like any other cosmetic. Generated entries themselves can be re-tinted from their own Variants tab.

Save / Load

Every edit autosaves to tools/cosmetics/cosmetics.json via the local Node server — no picker, no prompts. The toast in the bottom-right confirms each save. On next page load the calibrator reads from the same file.
